COVID-19 Predictive Modeling (Big Data & Machine Learning)

 Overview
A predictive model built using Machine Learning and Big Data techniques to analyze and forecast COVID-19 cases in Brazilian cities.

 Objectives
- Analyze large-scale COVID-19 data
- Clean, preprocess, and visualize data
- Apply predictive ML algorithms
- Generate trend and forecast visualizations

 Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn
- Hadoop / Apache Spark

 Project Workflow
1. Data collection & cleaning  
2. Exploratory Data Analysis (EDA)  
3. Feature engineering  
4. ML modeling  
5. Performance evaluation  

 Results
- Visualized trends of infections and deaths
- Built a predictive model based on historical data
