 Heart Disease Prediction Using Machine Learning

 Overview
A machine learning classification model that predicts the likelihood of heart disease based on medical attributes.


 Objectives
- Analyze medical dataset (UCI Heart Disease)
- Clean and preprocess data
- Handle outliers and missing values
- Train ML classifier (Random Forest)
- Build a simple web interface using Flask

 Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Flask (Web App)

 Project Workflow
1. Data analysis & visualization  
2. Feature engineering  
3. Model training & validation  
4. Evaluation using accuracy, recall, sensitivity  
5. Flask web interface  

 Results
- Achieved high accuracy with Random Forest classifier
- Built a usable web page for live prediction
